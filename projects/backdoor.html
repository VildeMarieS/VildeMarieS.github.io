Python Backdoor

A custom client/server backdoor written in Python using raw socket programming — built as part of a hands-on ethical hacking lab inside a controlled virtual machine environment.

Type Lab / Cybersecurity
Language Python
Environment Kali Linux + Windows VM
Course Zero to Mastery – Cybersecurity
screenshot / demo GIF goes here

What it does

The backdoor consists of two Python scripts: a server running on a Kali Linux attacker machine and a client planted on a Windows target VM. Once a connection is established, the server can send shell commands and receive the output in real time.

How it was built

  • Used Python's socket module to set up a raw TCP connection between the two VMs.
  • Configured both machines on a VirtualBox internal network with static IPs.
  • Implemented a persistent connection loop so the backdoor reconnects if dropped.
  • Bypassed Windows Defender interference during testing by adjusting exclusion paths.

What I learned

This lab gave me hands-on experience with low-level networking concepts — how sockets work, how commands travel over a connection, and how defenders detect this kind of traffic. Debugging the VM networking (internal vs. NAT adapters, static IPs via /etc/network/interfaces) was a challenge in itself and taught me a lot about Linux network configuration.
